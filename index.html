Open the console to experiment with frontend WebSocket clients and servers.

Examples
Create a WebSocket Server by making a new cSocketServer, with a custom address and response list.
The response argument is a large object, where the properties are functions named after their kind which return data to the clients connected
Create a WebSocket by making a new cWebSocket, with an address and onmessage list, where the onmessage list affects the DOM, updates data, etc.

Check out the JS (if you can stand the 8 character long tabs)
